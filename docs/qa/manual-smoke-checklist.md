# Ohh Lens Manual Smoke Checklist

- Launch the app and confirm the main window opens.
- Switch through Live, History, Files, and Setup.
- Start and stop listening.
- Change caption mode between original, translation, and dual-line.
- Confirm Setup shows backend and virtual-device guidance.
- Confirm a loopback device such as BlackHole appears when installed.
- Select `System audio`, choose the loopback device, and start listening.
- Play YouTube audio through the routed output and confirm Live changes from waiting to `Audio detected`.
- Stop playback and confirm the app returns to a waiting/silent state.
- Confirm History can show a saved preview transcript.
